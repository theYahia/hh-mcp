# /job-search — Поиск вакансий с аналитикой

## Алгоритм

1. Вызови `search_vacancies` с ключевыми словами, регионом, зарплатой
2. Вызови `get_salary_stats` для статистики зарплат по этой специальности
3. Покажи топ вакансий + медианную зарплату

## Формат ответа

```
## Вакансии: Python разработчик в Москве

Найдено: 1 234 вакансии
Медианная зарплата: 250 000 ₽

### Топ-5 вакансий
1. Senior Python Developer — Яндекс — 300 000–400 000 ₽
2. ...
```

## Примеры

```
/job-search Python Москва 200000
/job-search DevOps Петербург
```
